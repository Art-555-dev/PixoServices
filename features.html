Features

Here are the features of Pixo Programming Language:

    let (name) = (value) - Creates a variable (it also supports math expressions).

    print (message / variable) - Prints a message or variable to the console.

    ask '(message)' -> (variable) - Gets the user input and saves it, if '->' is used.

    if x > 5 - Starts an 'if' block. In this example, if x is larger than 5, it will execute code inside that block. If not, it will just continue.

    func (name): - Creates a new function which is saved in memory and that can be executed later when needed.

    forever: - Starts a loop. Press 'Ctrl + C' to stop the loop. If you want to run the loop, type 'end' and press 'Enter' on your keyboard.

    run function (name of function): - Runs a function.

    run file (name of file): - Runs a '.pixo' file.

    # (text) - Adds a comment which is ignored by Pixo Programming Language.

    clear - Clears the screen.

    exit - Closes Pixo Programming Language.
